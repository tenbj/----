# 健康养生研究方法论（专项）

本文档适用范围：**健康 / 减肥 / 养生 / 中医调理**类话题。
当话题落入此范围时，本方法论**完全覆盖**通用研究方法论（`研究方法论.md`）的第步论证部分，文档结构也改用下方七段式。

---

## 一、适用判断

满足以下任一条件即适用：

- 话题涉及身体症状、体质调理、体重管理、睡眠、情绪、饮食习惯
- 话题明确提到中医、西医、养生、减肥、保健、慢性病日常管理
- 话题的核心诉求是"怎么改善某个身体状态"

---

## 二、文档结构：七段式

```
一、用户完整问题（原文逐字记录，不得改写）
二、正面回答（一句话核心结论）
三、中医体系分析（主体）
四、西医理论体系分析（对照）
五、中西医比对（异同 + 相互印证）
六、生活最小改动方案
七、参考文献
```

> 健康 / 减肥 / 养生 / 中医调理类话题必须额外执行来源核验强制门。所有核心资料必须可访问、可验证或可下载，并说明当前权威性依据。

---

## 三、各段写作规范

### 3.1 一、用户完整问题

- **逐字引用**用户原始提问，不缩写、不改写、不归纳替代
- 若用户问题较长，全文保留

### 3.2 二、正面回答

- 一句话给出核心结论，不绕弯子
- 不在此处展开论证

### 3.3 三、中医体系分析（主体）

这是整篇文档的核心论证章节，权重最高。

**必须包含**：

1. **病机定位**：从阴阳、五行、脏腑、气血津液、经络等角度定位核心病机
2. **经典出处**：定位《黄帝内经》（素问 / 灵枢）、《难经》、《伤寒论》等经典中与话题直接相关的原文，**逐字引用**，注明篇名
3. **推导链条**：从经典原文出发，逐步推导到当前症状或问题的解释，每一步标注【合理推导】
4. **体质/时令补充**（如相关）：结合四时养生、体质辨识（九种体质）等补充说明

**禁止**：
- 笼统说"中医认为……"而不引用具体经典原文
- 推导跳层（直接从病机跳到结论，中间步骤省略）

**标注规范**：
- 经典原文 → 【经典出处】
- 逻辑延伸 → 【合理推导】

### 3.4 四、西医理论体系分析（对照）

- 从现代医学视角解释同一问题（生理机制、激素、神经系统、营养学等）
- 引用学术期刊、指南或权威机构资料，标明作者/机构、年份、期刊名或发布机构、DOI/指南编号/版本号
- 医学指南、共识、标准必须说明当前是否仍有效或是否被新版替代
- 不要求与中医结论一致，如实呈现现代医学的认识
- 标注：权威论文 → 【权威研究】；逻辑延伸 → 【合理推导】

### 3.5 五、中西医比对

- 列出中西医分析的**异同点**
- 标注相互印证之处（两边结论一致或互补）
- 标注分歧之处（两边结论不同或无法对应），不强行调和，如实记录

格式建议：

| 维度 | 中医视角 | 西医视角 | 异同 |
|------|---------|---------|------|
| 核心机制 | … | … | 印证 / 分歧 |
| 关键因素 | … | … | … |

### 3.6 六、生活最小改动方案

**核心原则**：
- **最小改动优先**：找到改动幅度最小、但对问题影响最大的杠杆点
- **非必要不给药物建议**：仅在问题已超出生活方式调整范围、且有明确适应症时才提
- **可操作**：方案要具体到"每天 / 每周 / 怎么做"，不给泛泛建议

**推荐结构**：

```
（一）饮食调整（若相关）
（二）作息调整（若相关）
（三）运动 / 活动（若相关）
（四）情志调节（若相关）
（五）穴位 / 推拿等中医简易外治（若适合且简单易行）
```

每条建议：
1. 说清楚"为什么"——链接到中医或西医的分析结论
2. 说清楚"怎么做"——频率、时长、量
3. 标注依据来源（哪条第一性原理推出此建议）

**药物建议触发条件**（同时满足才可提）：
- 生活方式调整已明确不足（症状较重 / 病程较长）
- 有明确的经典方剂或现代药物适应症
- 必须注明"建议就医/在执业医师指导下使用"

### 3.7 七、参考文献

必须使用通用方法论中的资料卡片模板。格式与通用方法论相同：

**中医经典**：
```
《经典名·篇名》——引用原文的关键句
```

**现代学术期刊**：
```
作者姓, 名缩写. (年份). 论文标题. *期刊名*, 卷(期): 起止页码. DOI
```

**医学指南/机构资料**：
```
机构名. (年份/版本). 文件标题. 发布机构/指南库. 可访问路径. 当前有效性依据.
```

---

## 四、质量检查清单

在落盘前逐项确认：

- [ ] 用户原始问题是否完整保留（未改写）
- [ ] 中医分析是否有经典原文逐字引用（篇名精确）
- [ ] 推导链条是否逐步可追溯，没有跳层
- [ ] 西医分析是否有学术文献支撑
- [ ] 医学/健康资料是否可访问、可验证或可下载
- [ ] 指南、共识、标准是否说明当前有效性或新版替代情况
- [ ] 比对表是否如实记录了分歧（没有强行统一）
- [ ] 生活方案是否具体可操作（有频率/量/时长）
- [ ] 是否避免了非必要药物建议
- [ ] 参考文献是否完整

---

## 五、边界

- 不替代执业医师的诊断；文档末尾如涉及具体症状治疗，必须注明"仅供参考，建议就医"
- 不凭空编造经典原文——如检索不到精确原文，标注"暂未定位原文，待核实"
- 不在对话框中给出完整内容——所有实质性内容落盘到 `02_课题研究_v*` 文档
