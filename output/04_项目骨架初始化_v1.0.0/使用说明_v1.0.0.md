# 使用说明 · 项目骨架初始化

> 这个子项目只交付普通脚本，不新增 `环境初始化` Skill。

## 文件

- `初始化知识研究工作区_v1.0.0.ps1`：创建新知识研究工作区骨架。

## 一键初始化

在当前脚本所在目录执行：

```powershell
powershell -ExecutionPolicy Bypass -File ".\初始化知识研究工作区_v1.0.0.ps1" -TargetRoot "E:\my_project\新知识研究"
```

也可以显式指定源工作区：

```powershell
powershell -ExecutionPolicy Bypass -File ".\初始化知识研究工作区_v1.0.0.ps1" -SourceRoot "E:\my_project\知识研究" -TargetRoot "E:\my_project\新知识研究"
```

## 会生成什么

- `.agent/rules/`
- `.agent/skills/子项目管理/`
- `.agent/skills/版本控制备份/`
- `.agent/skills/记忆管理/`
- `.memory/对话记录/`
- `.memory/知识提炼/`
- `.memory/系统记录/rules-skills.md`
- `.memory/全局知识地图.md`
- `.history/`
- `input/`
- `output/`

## 不会复制什么

- 不复制当前 `output/` 里的已有研究成果。
- 不复制当前 `.history/` 的历史快照。
- 不复制当前 `.memory/对话记录/` 和 `.memory/知识提炼/` 的具体项目内容。

## 目标目录已存在

默认要求目标目录为空。若确认要合并到已有目录，可加 `-Force`：

```powershell
powershell -ExecutionPolicy Bypass -File ".\初始化知识研究工作区_v1.0.0.ps1" -TargetRoot "E:\my_project\新知识研究" -Force
```

`-Force` 只会覆盖同名骨架文件，不会删除目标目录中的其他内容。
