所有对话记忆必须存储在 .memory/ 目录下，禁止存放在其他位置。

每次 AI 回复后，必须执行 Skill：记忆管理，先定位当前话题对应的 `.memory/对话记录/{子项目文件夹名}.md`，再追加本轮摘要。

若对话记录文件不存在，必须先按 `记忆管理` Skill 的模板初始化，再追加本轮摘要。

涉及 Skill、Rule、脚本、记忆文件的修改时，也属于需要记录的对话，不能因为“只是改系统文件”而跳过。

每次创建或修改 output/ 中的文件后，必须执行 Skill：记忆管理，追加知识提炼。

全局知识地图由子项目管理 Skill 在新建子项目时自动维护，无需手动触发。
