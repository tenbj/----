# 安装后本地化检查清单

## 安装完成后必须做什么

1. 给迁入 skill 选一个中文目录名
2. 保留原始 `SKILL.md` 到 `references/原始技能说明.md`
3. 重写薄 `SKILL.md`
4. 重写中文 `agents/openai.yaml`
5. 补齐 `agents/`、`references/`、`scripts/`、`assets/`
6. 清理根目录噪音文件

## 薄化时的底线

正确做法：

- 薄的是 `SKILL.md`
- 不薄的是总知识量
- 长规则要拆到 `references/`

错误做法：

- 直接删掉原始长说明
- 只留下脚本，不补中文入口和 references 路由
- 把稳定业务真相塞回 skill 私有说明里

## 重写 `SKILL.md` 时至少保留

- 这项技能解决什么问题
- 先读哪些本地知识
- 固定动作
- 什么时候再读 references
- 边界

## 最后再看一遍

- 新 skill 有没有和当前项目规则冲突
- 有没有把原始长说明保住
- 有没有补齐 references 路由
- 有没有把可发现性做得比安装前更差
