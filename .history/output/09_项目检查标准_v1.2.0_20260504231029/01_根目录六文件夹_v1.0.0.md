# 标准点 #1 · 根目录六文件夹

> 检查：项目根目录下六个文件夹是否完整存在。修正：缺少则创建。

---

## 标准

项目根目录下必须存在以下六个文件夹：

| # | 文件夹 | 用途 |
|---|--------|------|
| 1 | `.agents/` | 规则（rules/） + 技能（skills/） |
| 2 | `.history/` | 版本控制备份 |
| 3 | `.memory/` | 对话记忆、系统记录、知识提炼、全局知识地图 |
| 4 | `output/` | 子项目产出 |
| 5 | `input/` | 参考资料 |
| 6 | `.temp/` | 临时文件 |

---

## 检查

- 逐一验证六个文件夹是否存在。
- 缺失则标记为 [FAIL]。

## 修正

- 缺少的文件夹 → 创建空文件夹。
