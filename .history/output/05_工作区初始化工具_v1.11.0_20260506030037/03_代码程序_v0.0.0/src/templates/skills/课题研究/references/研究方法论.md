# 研究方法论：从问题到结论的标准流程

本文档定义了课题研究的完整方法论，回答三个核心问题：什么是第一性原理、怎么从原理走到结论、产出的文档该长什么样。

---

## 一、整体结构：五段式

每一份研究文档按此顺序展开：

```
一、你的问题（原文引用 + 问题归纳）
二、正面回答（一句话核心结论 + 框架摘要）
三、第一性原理论证（逐条论证核心结论的每一个支撑点）
四、总结（一句话回答 + 第一性原理汇总表）
五、参考文献（所有引用的完整索引）
```

要点：
- "问题"章节必须引用用户原文，不是改写
- "正面回答"必须在进入论证前给出一个不绕弯子的结论
- 论证章节每个小节先出结论，再展开来源和推导

---

## 二、什么是第一性原理

### 2.1 定义

第一性原理是问题领域中**不可再简化**的基础事实或规律。它不是推导出来的，它本身就是推导的起点。

判定标准：
- 它能被"证伪"吗？如果不能（证伪它需要推翻更基础的公理），它可能就是第一性原理
- 去掉它，后面的所有结论还能成立吗？如果不能，它就是推导的基石

### 2.2 找到第一性原理的方法

**中医体系**：
1. 从《黄帝内经·素问》《黄帝内经·灵枢》等经典中定位和话题直接相关的原文
2. 判断这段原文描述的是"基础规律"还是"具体应用"
3. 基础规律（如"人以水谷为本""百者心之使也""脾藏意"）→ 第一性原理
4. 具体应用（如某个病症的治法）→ 不是第一性原理，是对第一性原理的推导

**现代科学体系**：
1. 找到话题领域的奠基性论文或综述（如 Nature/Science 的首次鉴定论文，如某激素的发现）
2. 判断它揭示的是"基本机制"还是"应用层面发现"
3. 基本机制（如 Ghrelin 的首次鉴定、多巴胺预测信号的发现）→ 第一性原理
4. 应用层面发现（如某类药物对激素的影响）→ 是对第一性原理的衍生推导

### 2.3 第一性原理 vs 合理推导

| | 第一性原理 | 合理推导 |
|---|---|---|
| 性质 | 不可再简化的基础事实 | 从基础事实出发的逻辑延伸 |
| 来源 | 经典原文 / 奠基性学术发现 | 逻辑推理 |
| 标注 | 【经典出处】或【权威研究】 | 【合理推导】 |
| 能否无引用 | 不能——必须有出处 | 可以在逻辑自洽的前提下推定 |

**举例**：
- "脾藏营，营舍意"《灵枢·本神》→ 【经典出处】，第一性原理
- "脾虚则营气不足，营气不足则意无根基"→ 【合理推导】，基于上一句的逻辑延伸
- "意乱则念头浮游不定，且因脾与食物的亲和性，最容易浮现的是食物念头"→ 【合理推导】，两级推导

---

## 三、合理推导的规则

### 3.1 推导必须是可追溯的

每一条【合理推导】都必须能说清楚它从哪条第一性原理出发。读的人应该能沿着"第一性原理→推导1→推导2→结论"的链条追溯回去。

### 3.2 推导不能跨层跳跃

错误示例：
> 脾虚 → 所以想吃甜的。

跳跃了太多层。正确示例：
> 脾虚（前提）→ 脾气运化不足 → 营气化生减少（推导1，基于"脾藏营"）
> → "意"的物质基础减弱 → 念头浮游不定（推导2，基于"营舍意"）
> → 因甘入脾，脾与甘味有生理亲和性 → 浮现的最可能是甜食念头（推导3）

### 3.3 多层推导需标注级数

当推导超过两级时，在文末第一性原理汇总表中，允许使用"P1→D1→D2→C"的链式标注，让读者能够看清整条推理链。

---

## 四、结论的要求

### 4.1 两种结论

**核心结论**：放在"正面回答"章节，一句话。不引用、不推导——就是答案本身。

**总结性结论**：放在"总结"章节，可选包含：
- 第一性原理汇总表（每条对应支撑来源）
- 完整的推导链可视化（如 ASCII 链条图）
- 一句话总结

### 4.2 结论不能超过论证范围

结论只能覆盖被论证过的部分。不能在结论中引入论证章节从未讨论的新内容。

---

## 五、参考文献的格式

### 5.1 中医经典

格式：
```
《经典名·篇名》——引用原文的关键句
```

示例：
```
《灵枢·大惑论第八十》——"目者，心之使也。心者，神之舍也。"
```

### 5.2 现代学术期刊

格式：
```
作者姓, 名缩写. (年份). 论文标题. *期刊名*, 卷(期): 起止页码.
```

示例：
```
Spiegel K, Tasali E, Penev P, Van Cauter E. (2004). Brief communication: Sleep curtailment in healthy young men is associated with decreased leptin levels, elevated ghrelin levels, and increased hunger and appetite. *Annals of Internal Medicine*, 141(11): 846-850.
```

### 5.3 附加信息

对特别关键的论文，在参考文献条目后附加一个括号注明被引用到的具体数据，以便核实。

示例：
```
（本文引用数据：Leptin ↓18%, P=0.04; Ghrelin ↑28%, P<0.04; 饥饿 ↑24%, P<0.01）
```

## 六、文档的命名

研究文档命名格式：`{NN}_{主题核心词}_v{MAJOR}.{MINOR}.{PATCH}.md`

如果同一主题有多个文档，用描述性前缀区分：
- `中医视角·不饿却想喝的完整解读.md`
- `中医视角·不饿却想吃的通用解决方案.md`

研究文档位于 `02_课题研究_v*` 内，必须带独立文件版本号，新建默认 `v1.0.0`。新增、删除研究文档，或任一研究文档版本升级时，同步 bump `02_课题研究_v*` 文件夹版本。
