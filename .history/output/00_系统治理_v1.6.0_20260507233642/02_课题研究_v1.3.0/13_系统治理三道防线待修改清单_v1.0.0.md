## 2. 第一轮执行顺序

确认后按以下顺序执行：

1. 读取本清单和完整方案，确认执行范围。
2. 对 `.system/standards` 执行 FOLDER 备份。
3. 修改 `workspace-spec.json` 和 `工作区骨架规格.md`。
4. 分别对 3 个 Rule 执行 CONFIG 备份并修改。
5. 对 `课题研究`、`记忆管理`、`版本控制备份` 三个 Skill 执行 FOLDER 备份并修改。
6. 跑最小验证：JSON 解析、文案一致性、BOM 检查。
7. PROJECT 备份 05 子项目。
8. 同步 `.agents` 和 `.system` 到 05 模板。
9. 重新封装 05 初始化 EXE。
10. 跑 05 模板/产物验证。
11. 更新 00、05 的目录、版本记录、对话记录、系统记录、知识提炼。
