## 2026-05-04 · 享乐性食欲的神经机制

**核心观点**：

- 享乐性食欲（Hedonic Appetite）与饥饿性食欲（Homeostatic Appetite）是两条完全独立的神经通路。前者由能量亏空驱动（下丘脑），后者由奖赏预期驱动（伏隔核 + VTA 多巴胺系统）。
- "不饿但想吃"是纯粹的享乐性食欲，与血糖水平、饥饿素等稳态信号无关。
- 多巴胺在**期待**时释放而非获得时释放，这是享乐性食欲的关键机制——看到食物比吃到食物更让人兴奋。

**关键框架/模型**：

- **两条通路区分模型**：饥饿性食欲（下丘脑摄食中枢）vs 享乐性食欲（伏隔核 + VTA 多巴胺）
- **视觉→奖赏三阶段流程**：眶额叶皮层识别 → VTA→伏隔核多巴胺释放 → 前额叶冲动控制博弈
- **超级刺激物理论**：可乐作为"高糖 + 咖啡因"的人造超级刺激物，劫持进化塑造的奖赏系统
- **冲动vs控制博弈模型**：伏隔核（冲动）与 dlPFC（控制）的竞争决定是否行动

**与其他话题的关联**：

- → 食欲产生机制：享乐性食欲是食欲产生机制中"感官因素"和"享乐需求"的深度展开
