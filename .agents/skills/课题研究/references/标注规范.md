# 标注规范：证据标签的使用规则

本文档定义了研究文档中每条论断应标注的类型。标签不可混用，每个段落中只能有一种标签（除非一个段落包含两个不同性质的来源）。

---

## 核心标签总表

| 标签 | 含义 | 使用条件 |
|------|------|----------|
| 【第一来源】 | 原典、原始论文、官方标准、法规原文、原始数据 | 有可核验原始出处；优先于二手转述 |
| 【经典出处】 | 中医、哲学、法学等原典逐字引用 | 能定位到具体经典、篇名或章节 |
| 【权威研究】 | 同行评议论文、系统综述、权威机构报告 | 有 DOI、期刊、机构、报告编号或其他可验证标识 |
| 【当前标准】 | 官方仍维护或未废止的标准、指南、规范 | 能查到当前有效状态、版本或维护页面 |
| 【合理推导】 | 从前述来源出发的逻辑延伸 | 必须说明推导链，不能伪装成事实 |
| 【待验证】 | 暂未找到足够可核验来源 | 不得作为核心结论依据 |
| 【临床观察】 | 可靠医家或临床经验总结 | 只作经验层事实，不作第一性原理 |

---

## 标签说明

### 【经典出处】

**使用条件**：
- 中医古籍的原文引用（逐字，非意译）
- 定位到具体篇名（如"《素问·宣明五气》"而非泛泛的"《内经》"）
- 是直接引用，不是后世注释者对该经典的解释

**格式**：
```
**【经典出处】** 《素问·宣明五气》：
> "五味所入：酸入肝，辛入肺，苦入心，咸入肾，甘入脾。"
```

**不适用场景**：
- 对经典的现代解释、白话翻译 → 改为【合理推导】
- 古籍中描述了某种现象但未给出明确原文 → 不要标【经典出处】

### 【权威研究】

**使用条件**：
- 现代学术期刊发表的经过同行评议的论文
- 作者、年份、期刊可查
- 研究方法和结论经得起推敲

**格式**：
```
**【权威研究】** 作者姓, 名缩写. (年份). 论文标题. *期刊名*, 卷(期): 页码.
```

**适用类型**：
- 奠基性发现（如 Ghrelin 的首次鉴定）
- 系统性综述（如瘦素抵抗的分子机制综述）
- 高质量临床或实验研究（如 Spiegel 2004 睡眠与食欲激素研究）
- 心理学经典实验（如 Wegner 1987 白色熊实验）

**不适用场景**：
- 科普文章、博客、普通网站 → 不算权威研究
- 预印本（未经同行评议）→ 需要注明"预印本"
- 网络百科内容 → 不算权威研究，可以作为复习线索但仍需通过原文核实

**争议性研究**：
如果某项研究存在学术争议（如自我损耗理论在 2016 年多中心复制中未获支持），必须注明争议状态，不可将其作为确定无疑的第一性原理使用。

### 【第一来源】

**使用条件**：
- 直接来自原始出处，而不是二手转述
- 能给出可访问路径、可验证标识或可下载路径
- 可以是经典原典、原始论文、官方标准、法规原文、原始数据集

**要求**：
- 必须在参考文献资料卡片中说明当前权威性依据
- 若只能找到二手转述，不得标为【第一来源】

### 【当前标准】

**使用条件**：
- 官方标准、指南、规范、技术文档或政策文件
- 能确认当前仍有效、仍维护、未废止或未被替代

**要求**：
- 标明版本号、发布日期、修订日期或有效状态
- 如果无法确认有效状态，只能标为【待验证】或背景参考

### 【合理推导】

**使用条件**：
- 从【经典出处】或【权威研究】出发的逻辑延伸
- 在逻辑上是自洽的
- 明确标注为推导（不伪造成事实断言）

**格式**：
```
【合理推导】[推导内容]
```

**要求**：
- 必须在推导前或推导后说明它基于哪个第一性原理
- 多层推导需要能追溯回去
- 不能把【合理推导】当作"万用标签"——如果一条论断既没有经典出处也没有权威研究，且无法从已知原理出发做逻辑推演，那就**不应该出现在文档中**。

**示例**：
```
【合理推导】脾虚则营气不足。营气是"意"的物质基础（基于《灵枢·本神》"脾藏营，营舍意"）→ 营气不足则"意"的物质基础薄弱 → 念头无根、浮游不定。
```

### 【临床观察】

**使用条件**（很少使用）：
- 来自可靠的中医临床经验总结
- 在历代医家著作中有明确记载
- 非个人猜测

**不适用场景**：
- "我听说过""常见""大家知道" → 不算临床观察
- 个人经验 → 不算

### 【待验证】

**使用条件**：
- 暂时找不到可访问、可验证或可下载来源
- 来源可能有价值，但当前无法确认

**限制**：
- 不能进入核心结论
- 不能作为第一性原理
- 必须在后续研究中优先补证或删除

## 标签的层级关系

```
【第一来源】 / 【经典出处】 / 【权威研究】 / 【当前标准】
     ──── 可作为推导起点，但必须有可核验资料卡片
     │
     └──→ 【合理推导】 ── 从上面两者出发的逻辑延伸
              │
              └──→ 【合理推导】 ── 第二层推导，须注明来源
                       │
                       └──→ 可以继续，但超过三层应反思是否跳跃了
```

【临床观察】独立于这个层级，它不是推导演绎的起点或过程，而是经验层面的事实陈述。

---

## 常见错误

| 错误 | 正确做法 |
|------|---------|
| 对经典的白话翻译标【经典出处】 | 正确引用原文后，白话解释标【合理推导】 |
| 对科普网站内容标【权威研究】 | 溯源找到原始论文后再标 |
| 【合理推导】后不说明基于哪个原理 | 必须能追溯到上级标签 |
| 一整个大段落只标一个标签，但里面混了出处引用和逻辑推导 | 按最小语义单元分开标注 |
